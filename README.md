# Petit Monde

Single-file French-learning caregiving game for early elementary learners.

## Run

Serve the folder with any static server and open `petit-monde-v3.html`.

Examples:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/petit-monde-v3.html`.

## Included

- Room-based care loop with needs, potty training, expeditions, events, traits, and language progression
- Avatar picker with personality/style differentiation
- Wardrobe system with unlockable clothes and post-potty underwear progression
- First-run onboarding tutorial
- PWA manifest and service worker for install/offline support
